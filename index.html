Projetos Recentes

Projeto Réplica Spotify

Nesse projeto utilizando React, JavaScript, HTML e CSS, criei uma réplica da plataforma de streaming de música Spotify, onde se encontram os 20 artistas mais ouvidos do Brasil e suas respectivas 20 músicas mais ouvidas.

Projeto Análise de Dados

Nesse projeto utilizei Python e o framewrok Jupyter, para criar um sistema que analisa um histórico de clientes fictício de uma empresa, na qual analisa os dados dessa empresa e cria gráficos sobre as vendas e o motivo pela perda de clientes.

Projeto I.A - Análise de crédito

Nesse projeto utilizei Python e o framewrok Jupyter, para criar um I.A que é treinada para realizar uma análise de crédito de clientes fictícios de uma empresa bancária, na qual analisa os dados desse novo cliente empresa e avalia o "score" de crédito desse cliente.

Projeto Conta Bancária fictícia

Nesse projeto utilizei Python, criando uma interface básica onde simula um banco que funciona de forma a criar diferentes tipos de contas como salário, poupança, etc., e suas características onde simula o saque e depósito de uma conta bancária.
